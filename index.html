Security behavior

Cookie values are wiped as soon as an action starts, and again on tab hide or page close. The worker should keep origin checks and rate limits enabled.

Backend note

Search and import actions are designed to be proxied through a worker. The worker template included in this repo gives you a place to connect the final Roblox-side import endpoint.
